# 📊 Udemy Courses Analysis Dashboard (Tableau)

This project presents an interactive Tableau dashboard analyzing Udemy course data to uncover trends, patterns, and insights across different domains.

---

## 🔍 Project Overview

The dashboard provides a comprehensive analysis of Udemy courses, focusing on:

- Course distribution (Free vs Paid)
- Subject-wise course availability
- Subscriber trends across categories
- Most popular courses based on enrollment

---

## 📈 Key Insights

- 📚 Web Development has the highest number of courses and subscribers  
- 💰 Paid courses dominate over free courses  
- 👨‍🎓 Technology-related courses attract the most learners  
- 🏆 Top courses identified based on subscriber count
